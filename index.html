01 / 執行摘要

為什麼台灣銀行必須現在行動

三個結構性訊號同時發生：全球市場成熟、台灣監管明朗、國際對手已下場。錯過這個窗口，2028 年再做將失去先發優勢。

01
市場已成熟
B2B 年增 733%

McKinsey 過濾後的真實 B2B 支付達 $226B，並以每年 7 倍速度成長。新加坡、香港已成亞洲主要發起地。

02
台灣監管明朗
2026 H2 預計上路

金管會已明確穩定幣由金融機構優先發行，參照歐盟 MiCA 框架。銀行將成為第一波合法發行主體。

03
對手已下場
JPM、Citi、Stripe

JPMorgan Kinexys 已在 Base 公鏈發行 JPM Coin，Circle CPN 招募 55 家金融機構。時間窗口正在關閉。

★ 核心戰略建議

「代收代付為先，代幣化存款為輔」。以「企業 USDC/USDT 代收代付 + 自動結匯」為首波產品，6–12 個月內可落地；中期（2027 以後）再推出新台幣穩定幣（TWDC）或代幣化存款。建議費率採 0.15–0.50% 階梯訂價，相對 SWIFT 的 $25–50/筆 具明確優勢。

02 / 市場規模

全球穩定幣真實規模解構

不同指標講不同故事。看數字之前，必須先知道這個數字代表什麼。

2.1三層指標的真相

穩定幣市場常被引用的數字大相逕庭，因為三個指標代表完全不同的故事。點擊下方 tab 切換不同視角：

$
流通市值：$305–312B
存量——有多少錢停泊在穩定幣中

這是最基礎的指標。從 2023 年初的 $120B，到 2025 年底的 $308B，年增率約 50%。反映市場「正在持有」的規模。

對銀行的意義：市值增長反映需求與信任，但不代表使用強度。光看市值會低估真實的交易活動。

!
年交易量：$45 兆美元
含投機、套利、DeFi、內部轉帳

這是最常被引用但也最容易誤導的數字。超越 Visa 全年 $14 兆的新聞標題，其實 99% 是非支付活動。

警告：銀行做市場評估時不應採用這個數字做為潛在市場規模。它包含了大量重複計算的金融活動。

✓
真實支付量：$390B
McKinsey 剔除雜訊後的真實經濟活動

McKinsey 與 Artemis Analytics 在 2026 年聯合研究，透過「標記已知實體、金額區間過濾、行為模式識別」三層篩選得出的真實支付數字。

關鍵洞察：真實支付僅佔毛交易的 0.87%。這個數字遠小於媒體標題，但更貼近實際可觸及的市場。

B
B2B 支付：$226B
法人間流動——銀行最關注的池子

這是台灣銀行應該聚焦的目標市場。佔真實支付的 60%，2024→2025 年增 733%。

滲透率：相對全球 B2B 支付 $1.6 千兆/年，穩定幣僅佔 0.01%。巨大的成長空間。

看到「穩定幣年交易 $45 兆，超越 Visa」的標題要格外警惕。其中 99% 是投機與自動化活動，真實的支付用途僅佔 0.87%——這是銀行業評估市場時最常踩的陷阱。

3.2為何 USDC 離開了 Tron？

這是理解兩幣分化的關鍵案例。點擊展開查看各項原因：

1 Tron 的非法交易聲譽
+

Chainalysis 報告指出 2023 年逾 45% 的非法加密交易發生在 Tron 上。Hamas、伊朗革命衛隊、北韓 Lazarus Group 等被美國制裁的組織大量使用 TRC-20 USDT 洗錢。Circle 身為美國持牌穩定幣發行商，無法讓自己的產品出現在被制裁實體的錢包裡。

2 孫宇晨的 SEC 訴訟
+

2023 年 3 月，SEC 起訴 Tron 創辦人孫宇晨操縱代幣市場、未註冊證券發行、付費名人推廣未揭露。Circle 身為準備上市（2025/6 IPO）的公司，無法跟一個被 SEC 積極追訴的創辦人的鏈深度綁定。

3 哲學衝突
+

USDC 的核心價值主張是「受監管、透明、可信」；Tron 的生態文化是「低摩擦、匿名、跨境」。這兩者在哲學上相互矛盾——想合規的用戶不會選 Tron，想用 Tron 的用戶不在乎合規。結果是 USDC on Tron 流通量長期微不足道。

4 流通量極度不成比例
+

USDC on Tron 流通量約 $340M，相較 USDT on Tron 的 $85B，差距約 250 倍。Tether 先發優勢讓 Tron 成為 USDT 的壟斷領地，Circle 投入資源搶市效益極低。

5 策略重心轉向
+

Circle 把資源重押在 Solana（高效能、合規友善）、Base（Coinbase 自家 L2）、CCTP（自有跨鏈協議）。這是「棄雞肋、聚焦王牌」的典型策略。

i 對台灣銀行的啟示

銀行若選擇與穩定幣合作或發行，必須考慮與哪個生態綁定。選 USDC 不必然正確，但失誤成本較低；選 USDT 市場更大，但合規風險明顯較高。最佳策略是雙幣支援，但以 USDC 為合規主軸。

08 / 產品規格

兩大核心產品的規格詳解

聚焦階段 1–2 的兩個關鍵產品，可直接進入專案規劃。

目標客群

  • 年營收 NT$50M–5B 的中小型出口商
  • 電子商務賣家（Amazon、Shopify、獨立站收款）
  • ODM/OEM 代工廠收美國品牌客戶尾款
  • SaaS / 數位服務業海外訂閱收入
  • 跨境電商賣家收新興市場客戶款

七大功能模組

  1. 多鏈收款地址自動生成：客戶登入網銀可為每筆交易/每個買家生成唯一的 ERC-20、Tron、Solana 收款地址
  2. 即時對帳：鏈上確認後自動 match 到對應發票/訂單
  3. 自動結匯：可選擇保留穩定幣、即時結匯 TWD、或匯率條件觸發
  4. KYT/AML 篩檢：每筆收款經 Chainalysis/Elliptic 比對黑名單
  5. API 整合：提供 RESTful API 給 SAP、Oracle、鼎新 ERP 直接對接
  6. 多層審批流程：企業可設定出款審批層級、額度、白名單
  7. 報表與稅務：自動產生稅務申報格式的月度報表
⚙技術架構

托管：Fireblocks / BitGo / Coinbase Prime（MPC 多方計算）
結匯：與本地加密 OTC（XREX）或國際做市商簽約
合規：Chainalysis KYT + 自家 AML 規則引擎
支援鏈：Ethereum、Tron、Solana（Tier 1）；Base、Arbitrum（Tier 2）
支援幣：USDC（主推）、USDT（配套）

目標客群

  • 有頻繁海外採購/付款需求的進口商
  • 跨國企業集團的內部資金調度
  • 中小企業 B2B 付款（取代 SWIFT 電匯）
